# Nhân Vật Lịch Sử Việt Nam

Trang web đã được chuyển chủ đề từ 'Các Môn Thể Thao' sang 'Nhân Vật Lịch Sử Việt Nam'.

Các thay đổi chính:
- Cập nhật `index.html`, `dang_ki.html` để phù hợp chủ đề mới.
- Tạo trang `danh_tuong.html` và 4 trang nhân vật mẫu trong `nhan_vat_*.html`.
- Thêm hình placeholder trong `images/nhan_vat/` (SVG).
- Thêm mẫu `report.md` để hỗ trợ thực hiện và nộp báo cáo nhóm. (Đã xóa `rubric.html` theo yêu cầu.)

Hướng dẫn nhanh:
- Mở `index.html` để kiểm tra trang chủ.
- Thêm ảnh thực tế vào `images/nhan_vat/` (thay các file SVG nếu muốn) và cập nhật đường dẫn.
- Nếu muốn đổi tiêu đề hoặc thêm nhân vật, tạo file HTML mới theo mẫu `nhan_vat_*`.

Cần giúp tiếp:
- Thêm nội dung chi tiết cho từng nhân vật (mình có thể làm thêm).
- Tối ưu ảnh/webp, thêm captions, và kiểm tra accessibility (alt tags, tabindex).

---

Nếu ok, mình sẽ tiếp tục: (1) thêm nội dung chi tiết cho từng nhân vật, (2) tối ưu ảnh và thêm chú thích/nguồn, (3) cập nhật `dang_ki.html` để lưu dữ liệu form (nếu bạn muốn).
